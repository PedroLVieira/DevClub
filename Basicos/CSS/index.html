CSS Básico: como usar style

Você pode aplicar CSS de três formas principais:

  • Inline: no próprio elemento via atributo style="...".
  • Interno: dentro do <head> em um bloco <style> (como nesta página).
  • Externo: em um arquivo .css referenciado com <link rel="stylesheet" href="...">.

Inline style

Exemplo de estilo inline: este texto usa style no elemento. 

<span style="color:#1e3a8a; font-weight:700;">Texto estilizado</span>

Seletores: elemento, classe e id

Parágrafo normal (seletor por elemento).

Parágrafo com classe .destaque.

Classe + id: o #importante tem prioridade sobre a classe na cor.

/* CSS */
p { color: #334155; }
.destaque { color: #0f766e; font-weight: 700; }
#importante { color: #1e3a8a; }

Box model (conteúdo, padding, borda, margem)

Sou uma caixa: tenho padding dentro, borda ao redor e margem por fora.
.box-demo {
    background: #f8fafc; /* conteúdo */
    padding: 16px;       /* espaço interno */
    border: 4px solid #cbd5e1; /* borda */
    margin: 12px 0;      /* espaço externo */
}

Pseudo-classes (hover, active, disabled)

.btn-demo { background:#22c55e; }
.btn-demo:hover { background:#16a34a; }
.btn-demo:active { transform: translateY(1px); }
.btn-demo:disabled { background:#a7f3d0; cursor:not-allowed; }

Externo: arquivo CSS

Para reaproveitar estilos, prefira um arquivo .css:

<link rel="stylesheet" href="styles.css">
/* styles.css */
body { font-family: system-ui; }
.card { background:#fafcff; border:1px solid #c7d2fe; }
