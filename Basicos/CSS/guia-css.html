Seletores CSS

Seletores dizem quais elementos no HTML receberão estilos.

  • Tipo/Tag: aplica pela tag (ex.: p, h1).
  • Classe: começa com ponto e é reutilizável (ex.: .botao).
  • ID: único na página, começa com # (ex.: #principal).
  • Atributo: baseado em atributos (ex.: input[type="email"]).
  • Pseudo-classes: estados (ex.: a:hover, :focus).
/* Exemplos */
p { color: #444; }
.button { background: #6b46c1; color: #fff; }
#menu { position: sticky; top: 0; }
input[type="email"] { border-color: #3182ce; }
a:hover { text-decoration: underline; }

Folha de estilo em CASCATA

A "cascata" define como o navegador decide qual regra vence quando há conflito.

  • Ordem de importância: !important > estilos inline > especificidade > ordem no arquivo.
  • Especificidade: ID > classes/atributos/pseudo-classes > tags.
  • Última regra vence: regras mais abaixo geralmente sobem na cascata.
/* Ordem e especificidade */
.card { color: #1a202c; }          /* Classe */
#destaque { color: #2b6cb0; }      /* ID vence classe */
#destaque { color: #e53e3e !important; } /* !important vence todos */

Cores no CSS

Você pode definir cores em vários formatos.

  • Hex: #FF5733, #333.
  • RGB: rgb(255, 87, 51).
  • HSL: hsl(12, 100%, 60%).
  • Com transparência: rgba(...), hsla(...).

Unidade de medidas CSS

Unidades definem tamanhos relativos ou absolutos.

  • Absolutas: px.
  • Relativas ao elemento: em (ao font-size do pai), rem (ao root).
  • Viewport: vw, vh (largura/altura da janela).
  • Percentual: % relativo ao contexto.
.titulo { font-size: 2rem; }  /* 2 × tamanho base */
.card { width: 90%; }         /* 90% da largura do contêiner */
.hero { height: 50vh; }       /* metade da altura da janela */

Textos

Propriedades para tipografia e leitura.

  • Fonte: font-family, font-size, font-weight.
  • Espaçamento: line-height, letter-spacing.
  • Transformação: text-transform, text-decoration, text-align.
.paragrafo {
  font-family: system-ui, -apple-system, "Segoe UI", Roboto, sans-serif;
  font-size: 1rem;
  line-height: 1.6;
  text-align: justify;
}
.titulo { text-transform: uppercase; letter-spacing: 0.04em; }

Padding e Margin

Padding é o espaço interno; margin é o espaço externo do elemento.

.box { padding: 16px; margin: 24px 0; }
/* Shorthand: top right bottom left */
.box2 { padding: 10px 20px 30px 40px; }
/* Eixos: vertical horizontal */
.box3 { margin: 8px 16px; }

Box-sizing

Define como largura/altura são calculadas.

  • content-box: padrão; largura não inclui padding/borda.
  • border-box: largura inclui padding e borda (mais previsível).
/* Recomenda-se para layouts */
* { box-sizing: border-box; }

Display

Controla como um elemento participa do fluxo.

  • block: quebra linha, ocupa largura disponível.
  • inline: não quebra linha, respeita conteúdo.
  • inline-block: inline porém com dimensões.
  • flex: layout flexível em eixo.
  • grid: layout bidimensional em linhas/colunas.
.row { display: flex; gap: 8px; }
.grid { display: grid; grid-template-columns: repeat(3, 1fr); gap: 12px; }
