Input no HTML

O elemento <input> é usado para coletar dados do usuário. O comportamento muda conforme o atributo type.

Boas práticas

  • Use <label for=""> apontando para o id do input.
  • Defina name (é o que vai no envio do formulário).
  • Use required, min, max, pattern para validação.
  • Prefira autocomplete para melhorar a experiência.
  • Use mensagens de ajuda com aria-describedby.
