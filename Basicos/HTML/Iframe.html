Iframe no HTML

O elemento <iframe> permite incorporar outra página (ou conteúdo externo) dentro da sua página.

O que é um iframe?

Iframe significa inline frame (quadro embutido). Ele cria uma “janela” dentro do documento atual e carrega uma URL dentro dela.

Quando usar

  • Incorporar mapas, vídeos, formulários ou páginas internas do seu projeto.
  • Mostrar conteúdo de terceiros (quando permitido), mantendo seu layout principal.

Quando evitar

  • Quando você precisa de controle total do conteúdo (muitos sites bloqueiam incorporação).
  • Quando pode prejudicar performance/SEO ou causar problemas de responsividade.

Sintaxe básica

<iframe
    src="https://example.com"
    title="Exemplo de iframe"
    width="600"
    height="400"
></iframe>

Atributos comuns: src (URL), title (acessibilidade), width/height (tamanho).

Exemplo prático (conteúdo interno)

A forma mais estável de aprender é apontar o iframe para um arquivo do seu próprio projeto (por exemplo, ./index.html, se existir).

Se index.html não existir na mesma pasta, altere o src para um arquivo válido do seu projeto.

Segurança: sandbox e permissões

Iframes podem representar riscos quando carregam conteúdo de terceiros. Por isso, use sandbox para restringir o que o conteúdo embutido pode fazer. 

<iframe
    src="https://example.com"
    title="Iframe com restrições"
    sandbox
></iframe>

Você pode liberar permissões específicas (com cuidado), por exemplo: 

<iframe
    src="https://example.com"
    title="Iframe com permissões específicas"
    sandbox="allow-scripts allow-forms"
></iframe>

Para recursos específicos (câmera, autoplay, etc.), use o atributo allow quando necessário: 

<iframe
    src="https://example.com"
    title="Iframe com allow"
    allow="fullscreen; autoplay"
></iframe>

Responsividade

Um jeito comum é envolver o iframe em um container e controlar o tamanho via CSS (por exemplo, largura 100%). Aqui no HTML, deixamos classes (iframe-wrapper e iframe) para o seu styles.css padronizar.

Sugestão (no styles.css): defina .iframe { width: 100%; height: 60vh; border: 0; }.

Dica: sempre defina title no iframe (acessibilidade) e prefira loading="lazy" quando possível.
