Explicando a tag <link>

O que é?

A tag <link> é usada para referenciar recursos externos (principalmente CSS e ícones) e deve ficar dentro do <head>.

Ela é uma tag vazia (não tem fechamento), ou seja: <link ...>.

Diferença entre <link> e <a>

  • <link>: conecta recursos ao documento (não aparece como link na página).
  • <a>: cria um link clicável para navegação.

Atributos mais comuns

  • rel: descreve o tipo de relação (ex.: stylesheet, icon).
  • href: URL/caminho do recurso externo.
  • type: tipo do arquivo (MIME), útil em alguns casos (ex.: ícone).
  • media: define quando aplicar (ex.: estilos para impressão).

Exemplos

1) Importar CSS externo

<link rel="stylesheet" href="styles.css">

2) Definir favicon

<link rel="icon" href="favicon.ico" type="image/x-icon">

3) CSS específico para impressão

<link rel="stylesheet" href="print.css" media="print">

Resumo

Use <link> no <head> para “conectar” recursos externos ao seu HTML, principalmente arquivos de estilo e ícones.
