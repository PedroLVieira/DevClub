Formulários em HTML

Um formulário (<form>) coleta dados do usuário e envia para um servidor (ou para um script) através de um método (GET ou POST).

1) O elemento <form>

O <form> envolve os campos e define como os dados serão enviados. Principais atributos:

  • action: URL de destino (endpoint) que receberá os dados.
  • method: GET (dados na URL) ou POST (dados no corpo).
  • autocomplete: sugere preenchimento automático (on/off).
  • enctype: tipo de codificação (ex.: upload de arquivo usa multipart/form-data). 
<form action="/enviar" method="post" autocomplete="on">
    ...
</form>

2) Acessibilidade: label + id

Sempre associe <label> ao campo via for + id. Isso melhora acessibilidade e usabilidade. 

<label for="email">E-mail</label>
<input id="email" name="email" type="email" />

Para upload, o <form> usa enctype="multipart/form-data".

5) Dicas rápidas

  • Use required, minlength, pattern para validação básica.
  • Prefira type correto (email, tel, date, etc.).
  • Use name em todos os campos: é ele que “vira” dado enviado.
  • Organize com <fieldset> e <legend>.
