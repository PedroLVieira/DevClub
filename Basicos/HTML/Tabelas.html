Tabelas no HTML (<table>)

Use tabelas para dados tabulares (linhas e colunas). Evite usar tabela para layout.

Estrutura básica

  • <table>: elemento principal da tabela
  • <caption>: título/descrição da tabela (ajuda acessibilidade)
  • <thead>: cabeçalho
  • <tbody>: corpo (dados)
  • <tfoot>: rodapé (resumos/totalizadores)
  • <tr>: linha (table row)
  • <th>: célula de cabeçalho (table header)
  • <td>: célula de dado (table data)

Boas práticas

  • Use <th> para cabeçalhos e o atributo scope (col ou row) para melhorar a leitura por leitores de tela.
  • Prefira <caption> para descrever o propósito da tabela.
  • Para células que ocupam mais espaço: colspan (colunas) e rowspan (linhas).
  • Evite tabelas para montar layout; use CSS (Flexbox/Grid) para isso.

Modelo mínimo (para copiar)

<table>
    <caption>Título da tabela</caption>

    <thead>
        <tr>
            <th scope="col">Coluna 1</th>
            <th scope="col">Coluna 2</th>
        </tr>
    </thead>

    <tbody>
        <tr>
            <th scope="row">Linha 1</th>
            <td>Dado</td>
        </tr>
    </tbody>

    <tfoot>
        <tr>
            <th scope="row">Resumo</th>
            <td>Valor</td>
        </tr>
    </tfoot>
</table>
