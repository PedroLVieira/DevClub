Seletores do DOM

Um guia prático sobre como selecionar elementos no HTML com JavaScript.

Conteúdo: getElement* + querySelector*
Nível: básico → intermediário

Atalhos

Visão geral getElementById getElementsByClassName getElementsByTagName getElementsByName querySelector querySelectorAll Resumo Boas práticas Mini-exercício

Dica rápida

Você pode selecionar no documento inteiro (document) ou dentro de um elemento: 

// Busca no documento todo
const btn = document.querySelector('#btnSalvar');

// Busca apenas dentro de um container
const container = document.querySelector('.container');
const linksDoContainer = container?.querySelectorAll('a');

Visão geral

Seletores do DOM são funções que procuram elementos HTML para você ler/alterar conteúdo, trocar classes, capturar inputs, adicionar eventos, etc.

Família getElement*
Ótima para seleções simples e diretas (id, class, tag, name).
Família querySelector*
Super flexível: aceita qualquer seletor CSS (inclui combinadores, atributos, etc.).
Regra de ouro: Se pode não existir, trate null para não quebrar o código.

document.getElementById(id)

retorna 1 elemento

Retorna um único elemento que possua o atributo id igual ao valor informado.

  • Assinatura: HTMLElement | null
  • Boa prática: id deve ser único na página
  • Usado quando você já tem um elemento “principal” para capturar (botão, input, container específico, etc.)

Exemplo HTML

<button id="btnSalvar">Salvar</button>

Exemplo JS

const btn = document.getElementById('btnSalvar');

if (btn) {
  btn.addEventListener('click', () => {
    console.log('Cliquei em salvar');
  });
}

document.getElementsByClassName(className)

retorna coleção

Retorna uma coleção de elementos que tenham a classe informada.

  • Retorno: HTMLCollection (parece array, mas não é)
  • Importante: coleção live (se o DOM muda, ela muda)
  • Para usar map/filter, converta: Array.from(colecao) ou [...colecao]

Exemplo HTML

<li class="item">A</li>
<li class="item">B</li>

Exemplo JS

const itens = document.getElementsByClassName('item');

for (const el of itens) {
  el.classList.add('ativo');
}

document.getElementsByTagName(tagName)

retorna coleção

Retorna uma coleção (HTMLCollection) de todos os elementos com a tag informada.

  • Retorno: HTMLCollection (live)
  • Você pode usar 'div', 'a', 'p', etc.
  • '*' seleciona todas as tags (use com cuidado)

Exemplo JS

const links = document.getElementsByTagName('a');

for (const a of links) {
  a.setAttribute('target', '_blank');
}

Quando faz sentido?

Quando você quer aplicar uma regra para todas as tags de um tipo (ex.: todos os links, todos os parágrafos).

document.getElementsByName(name)

retorna lista

Retorna uma NodeList de elementos que possuam o atributo name com o valor informado.

  • Retorno: NodeList (normalmente estática)
  • Muito usado em formulários (radios/checkboxes com mesmo name)

Exemplo HTML

<input type="radio" name="cor" value="vermelho" />
<input type="radio" name="cor" value="azul" />

Exemplo JS

const radios = document.getElementsByName('cor');

radios.forEach((radio) => {
  radio.addEventListener('change', (e) => {
    const alvo = e.target;
    console.log('Mudou para:', alvo.value);
  });
});

Em alguns casos você pode checar se é um input: radio instanceof HTMLInputElement.

document.querySelector(selector)

retorna 1 elemento

Retorna o primeiro elemento que corresponda ao seletor CSS informado.

  • Assinatura: Element | null
  • Aceita qualquer seletor CSS: #id, .classe, tag, [atributo], combinadores, etc.
  • Se você quer vários, use querySelectorAll

Exemplos

const btn = document.querySelector('#btnSalvar');
const primeiroItem = document.querySelector('.item');
const inputEmail = document.querySelector('input[name="email"]');

Exemplo com segurança

const titulo = document.querySelector('h1');

if (titulo) {
  titulo.textContent = 'Título alterado via JS';
}

document.querySelectorAll(selector)

retorna lista

Retorna todos os elementos que correspondam ao seletor CSS informado.

  • Retorno: NodeList (estática)
  • Normalmente tem .forEach(), ótimo para iterar
  • Funciona com seletores CSS avançados

Exemplo

const cards = document.querySelectorAll('.card');

cards.forEach((card) => {
  card.classList.add('destacar');
});

Exemplo com eventos

const botoes = document.querySelectorAll('button');

botoes.forEach((btn) => {
  btn.addEventListener('click', () => console.log('Clique!'));
});

Resumo (comparação rápida)

Método Retorno Seleção Observações
getElementById HTMLElement | null id Um único elemento
getElementsByClassName HTMLCollection class Live collection (atualiza com o DOM)
getElementsByTagName HTMLCollection tag Live collection; cuidado com *
getElementsByName NodeList name Comum em inputs de formulário
querySelector Element | null seletor CSS Primeiro match
querySelectorAll NodeList seletor CSS Todos os matches; ótimo com forEach

Boas práticas e erros comuns

  1. Trate null quando o elemento pode não existir. 
    const el = document.querySelector('.nao-existe');

if (!el) {
  // Evita erro do tipo: Cannot read properties of null
  return;
}
  2. Entenda HTMLCollection vs NodeList.
    • HTMLCollection é live
    • NodeList geralmente é static
  3. Não confunda singular e plural.
    • getElementById (singular)
    • getElementsBy* (plural → coleção)
  4. Converta coleção para array quando necessário. 
    const colecao = document.getElementsByClassName('item');
const array = [...colecao];

// Agora dá para usar map/filter/reduce
const textos = array.map((el) => el.textContent);
  5. Prefira querySelector/querySelectorAll quando precisar de:
    • Seletores complexos
    • Seleção por atributo (ex.: input[name="email"])
    • Combinações (ex.: .menu a.ativo)

Mini-exercício (opcional)

Crie os itens abaixo no HTML e, depois, rode o JS para alterar o texto de cada item.

HTML

<ul>
  <li class="item">Item 1</li>
  <li class="item">Item 2</li>
</ul>

JS

const itens = document.querySelectorAll('.item');

itens.forEach((el, i) => {
  el.textContent = `Item ${i + 1} (alterado)`;
});
Desafio: Adicione uma classe CSS nos itens pares usando :nth-child(even) ou via JS com if ((i + 1) % 2 === 0).
