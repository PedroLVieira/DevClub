Listas no HTML

Em HTML existem dois tipos principais de listas: desordenadas (<ul>) e ordenadas (<ol>). Em ambas, cada item fica dentro de <li> (list item).

1) Lista desordenada (UL)

Use <ul> quando a ordem dos itens não importa. O navegador costuma mostrar marcadores (bolinhas).

  • HTML
  • CSS
  • JavaScript
<ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
</ul>

Você pode alterar o estilo do marcador via CSS com list-style-type:

  • Marcador quadrado
  • Outro item

2) Lista ordenada (OL)

Use <ol> quando a ordem dos itens importa (passo a passo, ranking, instruções). O navegador costuma numerar.

  1. Abrir o editor
  2. Criar o arquivo HTML
  3. Salvar e abrir no navegador
<ol>
    <li>Abrir o editor</li>
    <li>Criar o arquivo HTML</li>
    <li>Salvar e abrir no navegador</li>
</ol>

Você pode controlar a numeração com os atributos abaixo:

  • type: define o tipo (1, A, a, I, i)
  • start: define em qual número/letra começa
  • reversed: faz a contagem regressiva
  1. Começa em C
  2. Depois D
  3. Depois E
  1. Último
  2. Meio
  3. Primeiro

3) Listas aninhadas (lista dentro de lista)

Você pode criar subitens colocando uma lista <ul> ou <ol> dentro de um <li>.

  • Front-end
    • HTML
    • CSS
    • JavaScript
  • Back-end
    • Node.js
    • Python
<ul>
    <li>
        Front-end
        <ul>
            <li>HTML</li>
            <li>CSS</li>
            <li>JavaScript</li>
        </ul>
    </li>
</ul>

Resumo

  • <ul>: lista sem ordem (marcadores)
  • <ol>: lista com ordem (numeração/letras)
  • <li>: item da lista
